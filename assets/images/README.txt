PANDUAN PENGGANTIAN LOGO

1. logo.png
   - Ganti file ini dengan logo perusahaan Anda.
   - Format: PNG (Transparan).
   - Ukuran rekomendasi: 512x512px (persegi).
   - Digunakan di: Navbar, Footer, dan Mobile Icon.

2. favicon.png
   - Ganti file ini dengan icon kecil untuk tab browser.
   - Format: PNG.
   - Ukuran rekomendasi: 32x32px atau 64x64px.

3. hero-logo.png (Opsional)
   - Jika ingin logo besar di halaman depan, tambahkan file ini.

Catatan: Pastikan nama file sama persis agar website mendeteksi otomatis.
