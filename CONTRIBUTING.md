# Contributing

## Workflow
- Fork and create a feature branch
- Make changes and add examples if relevant
- Ensure CI passes
- Open a Pull Request

## Guidelines
- Keep repository free of secrets
- Prefer small, focused changes
- Update README/examples when utilities change

## Development
- Local preview: `python -m http.server 8080 --bind 127.0.0.1`
- Publish: `scripts/publish.ps1`
